如何用hugo写博客

第一步：安装HUGO

在官网上下载，并把hugo.exe路径放到PATH中，运行以下代码：

1

                brew install hugo

查看HUGO是否安装成功

1

                hugo version

第二步：新建一个网站:

1

                hugo new site mysite

第三步：安装主题：

1

                cd mysite
1
2

                git init 
                git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke

第四步：添加要写的博客：

1

                hugo new posts/my-first-post.md

此时，你的博客已经创建好了

第五步：查看当前的博客效果：

1

                Hugo server -D
