JS函数执行的时机

先看一段代码

1
2
3
4
5
6

let i = 0
for(i = 0; i<6; i++){
    setTimeout(()=>{
      console.log(i)
    },0)
}

猜猜执行的结果是什么呢？

可能你已经知道了答案：6，6，6，6，6，6

可是把代码改一下呢？

1
2
3
4
5

 for(let i = 0; i<6; i++){
   setTimeout(()=>{
     console.log(i)
  },0)
}

小小的变化答案差别却很大：0，1，2，3，4，5

why?很多人初学的时候始终都不能理解第一段代码，官方为了迎合新人在每次生成一个i时将它保存下来，如是有了不同的结果
