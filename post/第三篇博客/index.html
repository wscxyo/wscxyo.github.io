浅析URL

URL分哪几个部分？

  1. https协议
  2. 域名/IP
  3. 端口号
  4. 路径
  5. 查询参数
  6. 锚

IP是什么？

IP是Internet Protocol(网际互连协议)的缩写，用于定位一台设备。通过Ping命令我们可以得到相应的IP

域名

域名可以说是一个IP地址的代称，目的是为了便于记忆后者；域名还可分为不同级别，包括顶级域名、二级域名，三级域名

DNS

DNS （Domain Name System 的缩写）的作用就是根据域名查出IP地址。
